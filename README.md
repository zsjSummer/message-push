### 通过 Bark 向 iPhone 推送自定义消息

通过 Java 实现了后台向苹果服务器发送推送消息，然后苹果服务器推送消息到iPhone

#### 使用
在 App Store 下载 Bark App，部署该后台服务，登陆后在页面中选择订阅频道。预置登录用户名：admin@message.push.com，密码：admin

#### 功能更新
1. 增加了定时任务抓取微博热搜并推送（2019.02.21新增）
